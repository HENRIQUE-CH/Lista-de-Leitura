# Lista-de-Leitura
 App para adicionar livros que esteja lendo.
 Você pode marcar os livros já lidos, apagar ou editar. Também tem a função de tirar foto da capa do livro. 
 
 App Feito em Raect Native no Expo e comunicando com banco de dados AsyncStorage, atraves do curso Meu Primeiro Aplicativo do Samurai Dev.
